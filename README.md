# Rock Paper Scissors AI (PyQt + Python)

## Overview
This project is an intelligent Rock Paper Scissors game built using Python and PyQt5.  
It features an AI opponent that adapts to player behavior using pattern recognition techniques.

---

## Features

- Graphical User Interface (PyQt5)
- Two AI Difficulty Levels:
  - Easy: Frequency-based prediction
  - Medium: Markov Chain-based prediction
- Real-time score tracking
- Interactive buttons (Rock, Paper, Scissors)
- Color-based result feedback
- Reset game functionality

---

## How It Works

### Level 1: Frequency-Based AI
The AI tracks the user's most frequently played move and counters it.

### Level 2: Markov Chain AI
The AI learns patterns in sequences of moves and predicts the next move based on previous transitions.

---

## Tech Stack

- Python
- PyQt5
- Collections (Counter)
- Random module
